# Exercícios práticos para entendimento de loop e enquanto

### Exercicios 1

Faça um exercício para leitura de dados de uma eleição:
| Código | Candidato | Nome |
| :--- | :--- | :---: |
| 1 | Candidato | X |
| 2 | Candidato | Y |
| 3 | Branco |
| 0 | Encerrar Votação|

Qualquer opção diferente anulará o voto
No final deverá set exibido o total de votos e o percentual de voto de todos candidatos

| Código | Produto | Valor |

| 1 | Hambúrguer | R$ 3,00 |
| 2 | Cheeseburger | R$ 2,50 |
| 3 | Fritas| R$ 2,50 |
| 4 | Refrigerante | R$ 1,00 |
| 5 | Milkshake | R$ 3,00 |
| 0 | Sair | |
